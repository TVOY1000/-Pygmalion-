---

## Contribution Principles

- Respect human dignity
- No coercive mechanics
- No hidden monetization
- No dark patterns
- No gamification that creates dependency

If a contribution violates these principles, it will not be accepted.

---

## Process

1. Fork the repository
2. Create a feature or research branch
3. Submit a Pull Request with clear explanation
4. Engage in discussion if needed

---

## Communication

- Use GitHub Discussions for ideas
- Use Issues for concrete problems
- Be respectful and constructive

This project grows through dialogue, not pressure.
