This is a working MVP — not a product.

Pygmalion MVP is a functional prototype used for research, reflection, and pilot practices. It demonstrates how non-material human value can be recognized without money, coercion, ratings, or gamification.

This MVP exists to be tested, discussed, and evolved — not consumed.

What the MVP Does

  • Record acts of conscious participation
  • Express and register gratitude without creating obligations
  • Work with non-monetary accounting markers (first contour)
  • Observe a non-material footprint of actions, not a score or ranking

There is no accumulation, no competition, and no conversion into value or status.

Two Formats — Online & Offline

Online MVP (React)

  • Individual reflection flows
  • Acknowledgment of participation and trust
  • Experimental interface for non-material accounting

Offline MVP (HTML / CSS / JS)

  • Group facilitation tool
  • Used in discussions, workshops, and pilot communities
  • Supports collective reflection without digital pressure

Both versions are intentionally minimal and transparent.

How It Is Used Today

  • Facilitation instrument for small groups
  • Research tool for testing post-economic hypotheses
  • Prototype for ethical social-technology design
  • Discussion medium, not a behavioral controller

What the MVP Is Not

  • Not a social network
  • Not a DAO or token system
  • Not a reputation or scoring mechanism
  • Not a productivity or KPI tool

The MVP deliberately avoids mechanics that undermine trust or human dignity.

How to Participate

  • Explore the source code on GitHub
  • Open an Issue with a question, idea, or use-case
  • Propose a pilot group or research scenario
  • Contribute to code, documentation, or translations

This MVP evolves through dialogue, not through “growth hacking”.
